# 推演记录库

本目录存放对 v6 体系结构的其他两组沙盘推演记录（非典型案例，但仍有参考价值）。

---

## 两组记录说明

### case_480任务批量测试

对 v6 体系结构进行的**大规模批量覆盖测试**，按 M6 任务库全量执行 480+ 个任务，分 GEN/MAP/SYS/LEGACY 四类系统性输出报告。

**优点**：覆盖面广，适合快速发现遗漏  
**局限**：以结论为主，部分任务推演链不如典型案例完整

| 文件 | 内容 |
|------|------|
| `沙盘推演_480任务报告_00_总览_auto.md` | 整体统计与结论分布 |
| `沙盘推演_480任务报告_01_GEN_auto.md` | GEN 类（通用性）任务结果 |
| `沙盘推演_480任务报告_02_MAP_auto.md` | MAP 类（映射完备性）任务结果 |
| `沙盘推演_480任务报告_03_SYS_auto.md` | SYS 类（系统性）任务结果 |
| `沙盘推演_480任务报告_04_LEGACY_E_auto.md` | LEGACY-E 旧版兼容性测试 |
| `沙盘推演_480任务报告_05_LEGACY_N_auto.md` | LEGACY-N 旧版兼容性测试 |
| `沙盘推演_480任务执行记录_auto.md` | 逐条执行记录 |
| `gen_480_test_records.py` | 记录生成脚本 |

---

### case_标准记录_auto

另一套按 M2 标准格式自动生成的**场景记录集**，以 G3/G7 相关场景为主要验证焦点。

**特点**：严格遵循 M2 A-E 五节格式，每条记录的证据链完整

| 文件 | 内容 |
|------|------|
| `沙盘推演记录_v6_sonnet3_auto.md` | 标准 M2 场景记录集（G3-R003、G7 系列等）|
